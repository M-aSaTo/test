GitHub Enterpriseインスタンスがインターネットに接続されている場合は、ローカルPCから下記のコマンドを実行して、Support Bundleの作成とアップロードを一度に行うことが可能です。

```
ssh -p122 admin@[hostname] -- 'ghe-support-bundle  -t 40128'
```
